# GasolinaIQ 🔍⛽

Plataforma inteligente de análisis, comparación y predicción de precios de gasolina en México.

## Descripción

GasolinaIQ centraliza los datos públicos de precios de la CRE, los enriquece con variables macroeconómicas y aplica modelos de Machine Learning para predecir precios y recomendar la mejor gasolinera según ubicación, precio y calidad de servicio.
